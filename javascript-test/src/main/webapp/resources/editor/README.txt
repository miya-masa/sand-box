エディタを使ってみる
１．jshint-eclipseのインストール
		eclipseのメニューバーヘルプ -> 新規ソフトウェアインストール -> 追加 -> ロケーションに
		http://github.eclipsesource.com/jshint-eclipse/updates/
		をコピペ。"次へ"を押下していきeclipseを再起動する(ダイアログが出る)
		再起動後、jshint-sample.jsを開く。たくさん警告が付いていればインストール成功
２．フォーマッタを設定する
		プロジェクト右クリック -> プロパティ -> JavaScript -> コードスタイル -> フォーマッターを開く
		プロジェクト固有の設定を可能にするをチェック
		(指示があるまではここでストップ！)

		新規ボタンを押下し、プロファイル名"sample"コンボボックスEclipse[ビルトイン]を指定し、
		OKボタンを押下する。
		プロパティ画面の -> JavaScript -> コードスタイル -> フォーマッターでインポートを選択
		プロジェクト直下にあるdev-js-formatter.xmlを選択
